Pastikan berkas plugin yang akan Anda unggah memiliki struktur konten seperti ini:

~~~ .no-highlight
nama-plugin.zip
 └── nama-plugin/
     ├── file-1.txt
     ├── file-2.txt
     ├── file-3.txt
     ├── file-4.txt
     └── …
~~~

Anda perlu menambahkan sebuah folder sebagai pembungkus untuk berkas-berkas plugin Anda, kemudian beri nama berkas `ZIP` tersebut dengan nama yang sama seperti nama folder tersebut.
