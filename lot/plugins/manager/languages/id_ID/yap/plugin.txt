Pastikan berkas plugin yang akan Anda unggah memiliki struktur konten seperti ini:

~~~ .no-highlight
plugin.zip
└── plugin/
    ├── file-1.txt
    ├── file-2.txt
    ├── file-3.txt
    ├── file-4.txt
    └── …
~~~

Anda perlu menambahkan sebuah folder akar untuk membungkus berkas-berkas plugin Anda, dan kemudian Anda perlu menamakan berkas `ZIP` tersebut dengan nama yang sama seperti nama folder tersebut.
